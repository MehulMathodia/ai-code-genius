## üöÄ Getting Started

### Prerequisites
- Python 3.10+
- Node.js 18+
- Google AI Studio API Key

### Installation
1. **Clone the repo:**
   `git clone https://github.com/MehulMathodia/ai-code-genius.git`
2. **Setup Backend:**
   - `cd backend`
   - `pip install -r requirements.txt`
   - Create a `.env` file and add `GEMINI_API_KEY=your_key_here`
   - `python main.py`
3. **Setup Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm start`
